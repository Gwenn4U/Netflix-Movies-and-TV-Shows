# 📺 Netflix Movies and TV Shows Analysis
A group project for the Data Analysis & Management course.

# 📌 Project Overview
This project explores the Netflix Movies and TV Shows dataset to uncover interesting patterns in digital content. We focused on understanding how content is distributed globally, the typical duration of movies, and how Netflix's library has grown over the years.

# 📊 Dataset
The dataset is sourced from Kaggle and includes a comprehensive list of all movies and TV shows available on Netflix. It contains attributes like:

Type: Movie or TV Show

Release Year: The year the content was originally released.

Duration: Total minutes for movies or number of seasons for TV shows.

Country: The primary country where the content was produced.

Rating: Content maturity ratings (e.g., TV-MA, PG-13).

# 🛠 Methodology
We followed a standard data science lifecycle to ensure the accuracy and clarity of our findings:

Data Cleaning & Quality Handling: Addressed missing values in key columns like director, cast, and country.

Data Formatting & Transformation: Converted date strings into usable formats and extracted temporal data (Year/Month).

Data Wrangling: Filtered and grouped data to focus on top-performing countries and content types.

Exploratory Data Analysis (EDA): Performed statistical analysis to find averages, common trends, and distributions.

Data Visualization: Developed clear, high-impact charts using Matplotlib and Seaborn.

# 📈 Key Insights & Findings
Content Mix: Across all major markets, Movies significantly outnumber TV Shows. India shows the strongest preference for movies.

Audience Attention: The majority of Netflix movies are optimized for a 90-110 minute duration.

TV Show Lifecycle: Over 70% of TV series on the platform consist of only one season, highlighting a high-churn content strategy.

Recent Growth: Netflix experienced a massive surge in content additions between 2017 and 2021, with the US leading the growth.
